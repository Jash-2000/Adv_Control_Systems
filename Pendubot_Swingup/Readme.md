# Pendubot Swingup Operation

A typical pendubot works as an underacuated system as only the shoulder joint is active. Thus, it is a non-linear system whose dynamics are solved in a lot of literature. The pendubot system looks as follows:
![pendubot_Image]([https://coecsl.ece.illinois.edu/pages/pendubot.html](https://coecsl.ece.illinois.edu/images/dwg.gif))

To run the simulation, simply download this folder and extract the folder "OptimTraj-master" to the current directory. Simply execute main_Pendubot.m
